Football Prediction Point System

Rules for predicting football match scores and goal scorers:

Prediction Type Points
Correct Full-Time Score 3
Correct Winning Team or Draw 1
Correct First Goal Scorer 3
Correct Anytime Goal Scorer (except the first goal) 1

Number of Weekly Predictions:

  • Users can predict from 3 to 6 games each week.
  • If there are Champions League matches available, users can make one additional prediction each week.

Monthly Points Calculation:

  • Sum up the points earned by the user from all predictions made in a month.
